
두 확률변수 $X, Y$의 결합확률함수 $f(x, y)$에 대한 기댓값 $E[g(X, Y)]$를 다음과 같이 정의한다.

$$E[g(X,Y)] = \sum \sum g(x,y)f(x,y)$$
$$E[g(X,Y)] = \int \int g(x,y)f(x,y)dxdy$$
위의 식은 이산확률변수일 때, 아래는 연속확률변수인 경우이다.

## 기댓값의 성질

기댓값의 성질은 결합 분포가 아닐 때와 동일하다.

- $E(a) = 0$
- $E(aX + bY) = aE(X) + bE(Y)$
- $E[g(X, Y) + h(X, Y)] = E[g(X,Y)] + E[h(X, Y)]$
#### 적률
- $X$와 $Y$의 $(i,j)$차 적률 : $E(X^iY^j)$
- $X$와 $Y$의 $(i,j)$차 중심적률 : $E[(X-m_X)^i(Y-m_Y)^j]$

> [!note]
> $X$와 $Y$가 독립이면 $E(XY) = E(X)E(Y)$가 성립하지만 역은 성립하지 않는다.

>[!warning]
>기댓값을 구하는 것은 단일분포와 동일하지만 적분이 빡세다.
>적분 쉽게 하는 법 알아야 함.
>
>그리고 $x, y$의 구간을 잘 정해야 한다. 둘 중 하나를 먼저 정하자.
>
>예를 들어 $0 < x < y < \infty$ 인 경우, 먼저 $0 < x < \infty$로 정하면 $x < y < \infty$로 정해진다.
>
>반대로 $y$ 먼저 정해도 된다.
>
>조건부 기댓값에서도...

# 조건부 기댓값

X = a가 주어졌을 때, Y의 조건부 기댓값은 다음과 같이 정의한다.

$$E[Y|X=a] = \mu_{Y|X=a} = \sum yf_{Y|X}(y|a)$$
$$E[Y|X=a] = \mu_{Y|X=a} = \int yf_{Y|X}(y|a)dy$$
분산도 동일하게 다음과 같이 정의한다.

$$E[(Y-\mu_{Y|X=a})^2|X=a] = \sigma_{Y|X=a}^2 = Var(Y|X) = E(Y^2|X=a) - \mu_{Y|X=a}$$
$$= \sum (y-\mu_{Y|X=a})^2f_{Y|X}(y|a)$$
$$= \int (y-\mu_{Y|X=a})^2f_{Y|X}(y|a)dy$$
>[!info]
>그냥 확률만 조건부확률이고 기댓값 구하는 건 똑같다.


- 결합분포를 이용해 다음처럼 나타낼 수도 있다.

$$\mu_{Y|X=a} = \sum\frac{yf(a,y)}{f_X(a)}$$
$$\mu_{Y|X=a} = \int\frac{yf(a,y)}{f_X(a)}dy$$

>[!warning]
>위 결합분포의 기댓값에서 범위를 잘 지정해야 했다.
>
>조건부 기댓값은 $X=1$처럼 조건이 생기므로, 만약 $0 < x < y< \infty$였다면 y의 범위는 $1 < y < \infty$로 정해진다.
